Data flow:
Browser → Flask (HTTP fetch/POST)
Flask → PowerShell/WMI (subprocess)
Workers → External APIs (urllib · async · cached)
Cache read/write (JSON files on disk)
Task Scheduler → Scripts (at login / daily)
All PowerShell calls run via subprocess.run() · results parsed as JSON
